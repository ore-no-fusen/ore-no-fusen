# 俺の付箋 (Ore no Fusen) 要求仕様書 Δ0.4

## プロジェクト概要
**「俺の付箋」**は、ローカル Markdown ファイルをデスクトップ上に貼り付けられた「付箋」のように扱うためのミニマリスト向けアプリケーションです。

## 主要機能
- **ステルス・オーケストレーター (Stealth Orchestrator)**:
  - ベースフォルダ内の全 `.md` ファイルを自動スキャンし、前回の位置・サイズ・色で復元。
- **高精度な配置復元 (Geometry Persistence)**:
  - DPIスケールに対応した論理座標保存により、再起動時のズレを防止。
- **背景カラーカスタマイズ (Background Colors)**: [NEW]
  - 右クリックメニューから付箋ごとに背景色を選択可能。
  - **イエロー (黄色)**: 標準的な付箋カラー。
  - **グリーン (緑色)**: 目に優しいナチュラルカラー。
  - **ブルー (青色)**: 冷静な印象のスカイカラー。
  - 選択した色は `backgroundColor` として各ファイルの Frontmatter に保存される。
- **改行・空白行の完全維持 (Blank Line Preservation)**: [NEW]
  - エディタで入力した空白行が閲覧モードでも正しく垂直スペースとして表示されるようレンダリングを最適化。
- **インライン・ダイレクト編集 (ダブルクリック)**: 
  - 閲覧モードの文章をダブルクリックするだけで編集モードへ。
- **重なり・重複の徹底排除 (Single Window Policy)**:
  - 1ファイル1ウィンドウを徹底。

## 技術スタック
- **Frontend**: Next.js 14, React 18, Tailwind CSS
- **Backend**: Tauri v2 (Rust)
- **Persistence**: localStorage (フォルダ), Frontmatter (配置・色), sessionStorage (フラグ)

## UI/UX の極み
- **フォント**: `BIZ UDPGothic` / `BIZ UDGothic` による高精度タイポグラフィ。
- **密度**: A6サイズ相当の超高密度レイアウト。
- **透過**: Tauri 透過機能を活用したデスクトップ統合デザイン。
